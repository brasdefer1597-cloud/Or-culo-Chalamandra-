## 🏗️ ARQUITECTURA: EL RÍO DE LA CLARIDAD (Blueprints)

La arquitectura sigue el flujo de un sistema hídrico donde la información se purifica:

1.  **Manantial (User Input):** La "Anomalía" entra vía `useOracle` hook.
2.  **Filtros de Grava (UI/Components):** Se clasifica y valida en la interfaz.
3.  **Remolino Central (API Proxy):** `/api/decode` procesa la entrada usando DeepSeek con ingeniería de prompts táctica.
4.  **Estanque de Distribución (Logic/Services):** Los resultados se fragmentan en códigos accionables.
5.  **Océano de Valor (CTA):** El usuario se integra a la red para continuar el ciclo.

### Estructura de Carpetas
- `/pages`: Rutas y API Endpoints.
- `/components`: UI modular organizada por dominios.
- `/hooks`: Lógica de estado y orquestación.
- `/services`: Comunicación con el mundo exterior.
- `/lib/data`: El banco de conocimientos tácticos.
- `/types`: Cimientos tipados.
