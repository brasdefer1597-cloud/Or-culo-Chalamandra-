## 🏗️ ARQUITECTURA: EL RÍO DE LA CLARIDAD (Blueprints)

La arquitectura de **Chalamandra Magistral** sigue el flujo de un sistema hídrico donde la información se purifica a medida que avanza:

1.  **Manantial (User Input):** La "Anomalía" entra al sistema de forma cruda.
2.  **Filtros de Grava (OracleForm):** Se clasifica según la herramienta y el plano de aplicación.
3.  **Remolino Central (OracleService):** El motor SRAP (Gemini/Local) procesa la entrada usando ingeniería de prompts táctica.
4.  **Estanque de Distribución (ResultPanel):** La información se fragmenta en códigos accionables y feedback gamificado.
5.  **Océano de Valor (CTA):** El usuario se integra a la red para continuar el ciclo.
